1️⃣ What is the difference between var, let, and const?
ans : var old way and function scoped. let and const are block scoped, but const value cannot be reassigned.
2️⃣ What is the spread operator (...)?
ans : It expands array or object values into individual elements.
Example: [...arr1, ...arr2].
3️⃣ What is the difference between map(), filter(), and forEach()?
ans : map() creates a new array by modifying items, filter() returns items based on a condition.
forEach() just loops through items but does not return a new array.
4️⃣ What is an arrow function?
ans : A short syntax for writing functions in JavaScript using =>.
Example: const add = (a,b) => a + b.
5️⃣ What are template literals?
ans : Strings written with backticks ` ` that allow variables and expressions using ${}.
Example: Hello ${name}.
